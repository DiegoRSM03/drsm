# DRSM Design System Skill

Project-specific design system for Diego Sanchez's portfolio website.

## Purpose

Ensures visual consistency across all UI components by enforcing:

- Shape language (sharp vs rounded duality)
- Color palette usage
- Typography patterns
- Animation conventions
- Layout standards

## Files

- `SKILL.md` — Complete design system documentation
- `QUICK-REFERENCE.md` — Copy-paste patterns for common elements

## Usage

This skill should be automatically applied when building any UI component in this project. Reference the guidelines before creating new components.

## Key Principles

1. **Sharp edges** for decorative elements (floating shapes, highlights, cursor)
2. **Rounded edges** for interactive elements (buttons, cards, inputs)
3. **Purple (#8B5CF6)** as primary accent
4. **max-w-7xl** container for all sections
5. **Framer Motion** for all animations (no CSS keyframes)
